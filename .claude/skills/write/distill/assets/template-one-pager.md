# One-pager template — 1 page, classic

Use for project pitches, intake requests, handoffs. Reader who sees only this page must still be able to act.

Rule: fits on one printed page. Doesn't fit? Cut.

---

## Template

```markdown
# <Punchy title that carries the finding, not the topic>

*<date> · <author/owner> · Status: <Draft | For review | Final>*

---

## What happened / What it is

<2-3 sentences. Anchored to a date and a named system or merchant.>

## Why it matters

| | |
|---|---|
| **<stakeholder impact>** | <quantified> |
| **<trust / merchant impact>** | <quantified> |
| **<pattern / systemic impact>** | <quantified> |
| **<documented action items / gaps>** | <count + status> |

## The gap, named

> **<1-sentence statement of the structural gap.>**

<1-2 sentences elaborating.>

## What's been done

- <completed item> — <link or evidence>
- <open item> — <status>

## The ask

**<N> things, ranked by leverage:**

1. **<action>** — <why highest leverage>. <owner + timing>.
2. **<action>** — <why>. <owner + timing>.
3. **<action>** — <why>. <owner + timing>.

## Receipts

- **<channel>:** <link>
- **<dashboard / Quick site>:** <link>
- **<Slack thread / message>:** <link with quoted snippet>
- **<Vault doc / GSD>:** <link>

---

*Drafted <datetime> for <author>'s review before any sharing.*
```

---

## Voice anchors

- Lead with "what happened" or "what it is" — not "background"
- 2-3 sentences in the opener, anchored to a real date and a real merchant or system
- "Why it matters" table = 4 rows, each row quantified
- "The gap, named" is one blockquoted sentence — the structural reading
- "What's been done" mixes closed and open items so the reader sees momentum
- The ask is ranked 1-2-3 with explicit owners and timing — not a wish list
- Receipts make verification one click away
- Status footer prevents premature sharing
- No "leverage" / "synergy" / "best-in-class" / "going forward"
- No emojis
- Tatiana title: Enterprise Technical Support Expert
- Claire Shiminsky named as co-lead on SRR / ES Triage work
- Real merchants: Omaze, Harry's, Colgate Palmolive, Estrid Studios, Alo Yoga
